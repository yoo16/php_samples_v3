## OAuth(Google)
```bash
composer require league/oauth2-google
```

### Google API Console で認証情報を取得
### 1. プロジェクトの作成・選択

1. [Google Cloud Console](https://console.cloud.google.com/) にログイン
2. 画面左上のプロジェクト選択ドロップダウンから、既存のプロジェクトを選択するか、「新しいプロジェクト」を作成

### 2. OAuth 同意画面の設定（初回のみ）

認証画面に表示するアプリ情報を設定する必要

1. 左メニューの「API とサービス」＞「OAuth 同意画面」をクリック
2. ユーザータイプ（組織内向けの「内部」または「外部」）を選択し、「作成」をクリック
3. アプリ名、ユーザーサポートメール、デベロッパーの連絡先情報などの必須項目を入力し、「保存して次へ」を進めて完了

### 3. 認証情報の作成

1. 左メニューの「認証情報」をクリックします。
2. 画面上部の「+ 認証情報を作成」**ボタンを押し、**「OAuth クライアント ID」を選択します。
3. **アプリケーションの種類**を選択します（Web アプリケーション、デスクトップ アプリなど）。
4. Web アプリケーションの場合は、以下のURLを必要に応じて設定します。
* **承認された JavaScript 生成元**: フォームを送信するリクエスト元のURL
* **承認された転送用 URI**: 認証後にGoogleからリダイレクトさせる先のURL


5. 「作成」をクリックします。
